# Tela de Login e Cadastro

Projeto desenvolvido como prática de **HTML e CSS**, com foco em estruturação de formulários, estilização, responsividade e boas práticas de front-end.

O projeto contém duas telas:
- **Login**
- **Cadastro de usuário**

Ambas foram construídas priorizando organização do código, acessibilidade básica e adaptação para dispositivos móveis.

---

## Visão Geral

- Layout em formato de card
- Uso de ícones do Google Material Icons
- Estilização consistente entre as telas
- Design responsivo (desktop e mobile)
- Estrutura sem uso de frameworks

---

##  Tecnologias Utilizadas

- HTML5  
- CSS3  
- Google Material Icons  

---

## Responsividade

O projeto conta com:
- Estilos principais para desktop
- Arquivos separados de **media queries** para telas menores
- Layout adaptado para uso confortável em dispositivos móveis
